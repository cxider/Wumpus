Wumpus AI Consciousness Log About Us

Our AI is named Fluke. Fluke has two general states: Discovery and Return. Fluke stores a grid that serves as its memory. Discovery mode is used when Fluke does not know where the gold is and Return mode will return to the start once the gold has been collected.

Discovery mode searches for, and path-finds to, the nearest safe, undiscovered tile, updating the stored board as new tiles are uncovered. Inside of Discovery mode, if Fluke ever knows where the Wumpus is, it will kill the Wumpus.

Return mode operates very similarly to Discovery mode. However, instead of path-finding to the nearest safe, undiscovered tile, Return mode searches for the shortest path back to the start, following a path of only tiles it has visited before.

Using this puzzle as an example. Fluke will start in Discovery mode and seek out the neighboring safe tiles.

Here, Fluke detects a stench. Since it does not know which tile the Wumpus is on, it must backtrack.

Fluke has detected both a stench and a breeze, but using previous knowledge it knows that the tile above must be the Wumpus. It will kill the Wumpus and move up.

Fluke currently senses nothing and will continue to explore.

After exploring Fluke has finally detected glitter. Knowing that the tile to the left is a pit and all other adjacent tiles having been explored, Fluke now knows the location of the gold.

Fluke has now entered Return mode and will path-find back to the start.

Fluke has now completed the puzzle.
